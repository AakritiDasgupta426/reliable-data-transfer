# Reliable Data Transfer Protocol

Implemented a reliable data transfer protocol over an unreliable network emulator.

## Overview

This project implements core networking protocols to ensure reliable communication over unreliable channels.

## Features

- Stop-and-Wait protocol baseline
- Sliding Window protocol for improved throughput
- Handles packet loss and reordering
- Performance evaluation under varying conditions

## Results

- Sliding window significantly improved throughput over stop-and-wait
- Evaluated goodput, overhead, and transmission time
- Performance tested under different packet loss rates and window sizes

## What I Learned

- Designing reliable communication protocols
- Tradeoffs between throughput and reliability
- Handling real-world network issues like loss and reordering
